 A Student Management System developed in **C++** and **Web Technologies** to manage student records efficiently through both console-based and graphical interfaces.

## 📖 About The Project

This project began as a **C++ Console-Based Student Management System** and was later transformed into a **Web-Based Application** using HTML, CSS, and JavaScript.

The system allows users to perform various student management operations such as adding, searching, updating, and deleting student records.

---

## ✨ Features

### 👨‍🎓 Student Management

* ➕ Add New Student
* 🔍 Search Student
* ✏️ Update Student Information
* 🗑️ Delete Student Records
* 📋 Display All Students

### 🌐 Web Interface

* Modern UI Design
* JIIT-Themed Dashboard
* Responsive Layout
* Interactive Search Functionality
* Dynamic Student Records Table
